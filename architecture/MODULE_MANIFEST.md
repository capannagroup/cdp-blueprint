# Module Manifest

## Purpose

The Module Manifest is the official registry of all business modules inside the Capanna Digital Platform (CDP).

Every module must be registered here before implementation.

Each module shall have:

- README.md
- Architecture document
- Business specification
- API specification
- Database entities
- Runtime implementation
