# Production Dashboard

## Overview

The Production Dashboard provides real-time visibility into manufacturing operations, allowing executives, production managers, supervisors, planners, and operators to monitor performance and make informed decisions.

The dashboard consolidates production, machine, operator, quality, maintenance, and inventory data into a single interface.
