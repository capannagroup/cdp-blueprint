# Work Centers

## Overview

Work Centers represent the physical or logical production resources where manufacturing operations are executed. A Work Center may represent a machine, production line, department, or manual workstation.

Every routing operation is assigned to a Work Center.

---

## Objectives

- Manage manufacturing resources
- Allocate production capacity
- Schedule machine workloads
- Track operational performance
- Balance production flow
- Optimize resource utilization

---

## Work Center Types

### Machine Work Centers

- Beam Saw
- CNC Router
- CNC 6-Side Drilling
- Edge Banding
- Sanding
- Painting Line

### Manual Work Centers

- Assembly
- Hardware Installation
- Inspection
- Packaging

### Logistics Work Centers

- Material Preparation
- Warehouse
- Shipping

---

## Configuration

Each Work Center contains:

- Work Center Code
- Name
- Department
- Machine Type
- Capacity
- Calendar
- Shift Pattern
- Efficiency
- Status
- Cost Rate

---

## Capacity

Capacity is calculated using:
