# Machines

## Overview

The Machines component manages all manufacturing equipment used within the factory. It stores machine specifications, capabilities, capacities, maintenance schedules, operating status, and production history.

Each machine belongs to a Work Center and may execute one or more manufacturing operations.

---

## Objectives

- Register manufacturing equipment
- Track machine utilization
- Monitor machine status
- Record machine performance
- Schedule preventive maintenance
- Improve Overall Equipment Effectiveness (OEE)

---

## Machine Profile

Each machine contains:

- Machine ID
- Machine Code
- Machine Name
- Manufacturer
- Model
- Serial Number
- Factory Location
- Work Center
- Installation Date
- Power Requirements
- Controller
- Software Version
- Status

---

## Machine Types

- CNC 6-Side Drill
- Beam Saw

- Sanding Machine

Each machine defines:
