# Shop Floor Control

## Overview

The Shop Floor Control component manages real-time production activities across all manufacturing work centers. It provides operational visibility, machine monitoring, labor tracking, production reporting, and execution management.

Shop Floor Control acts as the Manufacturing Execution System (MES) layer between Production Planning and Machine Operations.

---

## Objectives

- Execute production schedules
- Monitor machine activity
- Track operator performance
- Collect production data
- Report manufacturing progress
- Control production flow
- Improve manufacturing efficiency

---

## Work Centers

Typical work centers include:

- Beam Saw
- CNC Router
- CNC Drilling
- Edge Banding
- Sanding
- Assembly
- Painting
- Packaging
- Warehouse Transfer
