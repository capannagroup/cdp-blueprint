# Role Based Access Control (RBAC)

## Purpose

Role Based Access Control (RBAC) defines which users can access which parts of the Capanna Digital Platform.

---

## Standard Roles

- Super Administrator
- System Administrator
- Factory Manager
- Sales Manager
- Sales Representative
- Purchasing Manager
- Warehouse Manager
- Production Manager
- Finance Manager
- Accountant
- HR Manager
- Employee
- Customer
- Supplier
- External Auditor

---
