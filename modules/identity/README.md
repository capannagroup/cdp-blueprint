# Identity Module

The Identity Module provides centralized identity and access management across the Capanna Digital Platform.

## Responsibilities

- User Management
- Organization Management
- Tenant Management
- Team Management
- Roles
- Permissions
- Authentication Integration
- Identity Federation
- Account Lifecycle
- Invitations
- Profile Management
